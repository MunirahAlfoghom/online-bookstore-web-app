Online Bookstore Web Application

A full-stack web application for managing books using PHP and MySQL.

Features
- Add new books
- Edit book details
- Delete books
- View all books in a table
- Form validation using JavaScript

Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

 How to Run
1. Import the database.sql file into MySQL
2. Update config.php with your database settings
3. Run the project using XAMPP or any local server

Project Structure
- index.html (Home page)
- books.php (Display books)
- add_book.php (Add new book)
- edit_book.php (Edit book)
- delete_book.php (Delete book)
- database.sql (Database)
